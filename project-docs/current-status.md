# الحالة الحالية للمشروع

## آخر تحديث
- التاريخ: 2026-04-08
- المرحلة الحالية: رفع التمايز البصري بين أنواع الصفحات مع الحفاظ على الهوية العامة للموقع

## ما أُنجز حتى الآن
- بناء الموقع على Astro بهيكل عربي وإنجليزي.
- تشغيل الصفحات الأساسية: الرئيسية، النبذة، السيرة الذاتية، الكتب، التحقيقات، المقالات.
- إضافة أقسام مستقلة في الملاحة للآتي:
  - المجالات العلمية
  - المشروعات
  - الوسائط
  - الروابط الخارجية
  - التواصل
- إنشاء صفحة مستقلة للمسار التعليمي المقترح.
- إنشاء صفحات تمهيدية فرعية للمجالات العلمية المتخصصة بالعربية والإنجليزية.
- تفعيل التبديل بين العربية والإنجليزية للصفحات الأساسية والصفحات الفرعية الجديدة.
- توسيع `Content Collections` لتشمل:
  - الوسائط
  - الروابط الخارجية
  - المجالات العلمية
- إضافة ملفات محتوى أولية لهذه المجموعات الجديدة لتفعيل البنية المنظمة.
- تحديث الصفحة الرئيسية والفوتر ليعكسا الأقسام الموسعة ومسارات التوسع الجديدة.
- إضافة `breadcrumbs` موحدة للصفحات الداخلية.
- إنشاء صفحة دليل عام للموقع بالعربية والإنجليزية لتسهيل الوصول السريع إلى الأقسام.
- إعادة تنظيم الهيدر إلى طبقتين بصريًا:
  - روابط أساسية
  - روابط توسع ثانوية
- إضافة وصلة ظاهرة إلى دليل الموقع داخل الهيدر.
- تمييز بصري بين:
  - الصفحة الرئيسية
  - صفحات البوابات الجامعة
  - الصفحات الفرعية المتخصصة
- إضافة شارات وصفية خفيفة داخل الـ hero في عدد من الصفحات لتوضيح طبيعة كل مسار.

## الحالة الفنية
- آخر تحقق ناجح:
  - `npm run build`
  - `npm run check`
- النتيجة: لا توجد أخطاء أو تحذيرات.

## ما ينقص الآن
- إدخال المحتوى الحقيقي المعتمد بدل الصياغات التأسيسية.
- ربط صفحات المجالات العلمية بمقالات فعلية أو بيانات محتوى منظمة.
- إضافة دعم منظم لاحقًا للتصنيفات أو السلاسل أو صفحات البيانات الثابتة عند الحاجة.
- استبدال المداخل التجريبية الحالية ببيانات معتمدة فعلية.
- تحسين إضافي لاحق للهيدر إذا احتاج الأمر إلى تنظيم بصري أهدأ مع ازدياد الأقسام.
- اختبار فعلي لاحق على أجهزة مختلفة للتأكد من أن الإيقاع البصري للهيدر مريح بعد ازدياد الروابط.
- تعميم منطق التمايز البصري لاحقًا على صفحات أخرى مثل الكتب والمشروعات والوسائط إذا احتجنا مزيدًا من التخصيص.

## الخطوة التالية المقترحة
- البدء في إدخال المحتوى الحقيقي لأول قسمين أو ثلاثة أقسام ذات أولوية، أو
- الاستمرار في جعل الأقسام الجديدة مدفوعة بالبيانات المنظمة بدل النصوص اليدوية كلما أمكن.
