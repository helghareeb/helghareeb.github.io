# سجل التحديثات

## 2026-04-04
### ما أُنجز
- بدء نقل المشروع من الصفحة القديمة إلى موقع علمي شخصي جديد.
- اعتماد GitHub Pages منصة نشر للموقع.
- اعتماد التوجه العام: موقع حديث هجين بين الأكاديمي والتراثي.
- اعتماد البنية الثنائية للغات: العربية والإنجليزية.
- اعتماد خريطة الموقع الأولية ثم تحديثها وفق قرارات المشروع.
- اعتماد فصل الكتب عن التحقيقات.
- اعتماد أقسام مستقلة للمخطوطات، والدفاع عن السنة والشبهات، والذكاء الاصطناعي والتراث الإسلامي، والتعليم الإلكتروني للعلوم الإسلامية، والتحكيم العلمي، والمشروعات العلمية، والوسائط.
- اعتماد قسم مستقل للألوكة، مع إمكان التصنيف والسلاسل.
- اعتماد قسم المسار التعليمي المقترح لطالب العلم.
- اعتماد قسم الروابط الخارجية.
- اعتماد أن التحكيم العلمي سيعرض عرضًا مختصرًا عامًا للرسائل والأعمال المحكمة.
- رفع ملفات التوثيق الأساسية إلى المستودع داخل `project-docs/`.
