# سجل التحديثات

## 2026-04-08
### ما أُنجز
- مراجعة المستودع والوثائق المرجعية كاملة قبل استئناف العمل.
- تثبيت الفجوة بين خريطة المشروع المعتمدة والتنفيذ الحالي بوصفها أولوية التنفيذ التالية.
- توسيع الملاحة الفعلية في الموقع لتشمل:
  - المجالات العلمية
  - المشروعات
  - الوسائط
  - الروابط الخارجية
  - تواصل
- إنشاء الصفحات التأسيسية لهذه الأقسام بالعربية والإنجليزية.
- إنشاء صفحة مستقلة للمسار التعليمي المقترح بالعربية والإنجليزية.
- إنشاء بوابة `المجالات العلمية` وصفحة `Scholarly Fields` وربطهما بالملاحة.
- إنشاء الصفحات الفرعية الفعلية للمجالات العلمية المتخصصة بالعربية:
  - الدراسات الإسلامية
  - الدفاع عن السنة والشبهات
  - الألوكة
  - المخطوطات
  - الذكاء الاصطناعي والتراث الإسلامي
  - التحكيم العلمي
  - التعليم الإلكتروني للعلوم الإسلامية
- إنشاء الصفحات الفرعية المناظرة بالإنجليزية.
- ربط بطاقات المجالات العلمية بهذه الصفحات الفرعية.
- تحديث زر التبديل بين العربية والإنجليزية ليغطي الصفحات الجديدة، بما فيها صفحات المجالات الفرعية.
- إنشاء ملف حالة مرجعي مختصر داخل:
  - `project-docs/current-status.md`
  ليُحدَّث مع كل جولة مهمة من العمل.
- توسيع `Content Collections` داخل `src/content.config.ts` لتشمل:
  - `media`
  - `externalLinks`
  - `fields`
- إنشاء مداخل محتوى أولية لهذه المجموعات الجديدة لتفعيل البنية المنظمة من الآن.
- تحديث الصفحة الرئيسية العربية والإنجليزية بحيث تبرز:
  - المجالات العلمية
  - المشروعات
  - الوسائط
  - الروابط الخارجية
  إلى جانب الكتب والتحقيقات والمقالات.
- تحديث الفوتر ليعكس الأقسام الأساسية وأقسام التوسع بدل الاكتفاء بالبنية القديمة.
- إضافة مكوّن `breadcrumbs` موحد يظهر في الصفحات الداخلية لرفع وضوح التنقل.
- إنشاء صفحة `Site Index` / `دليل الموقع` في اللغتين كفهرس عام للبنية الحالية.
- ربط دليل الموقع بالفوتر وتحديث زر التبديل بين العربية والإنجليزية ليشمل هذه الصفحة أيضًا.
- إعادة تنظيم الهيدر بصريًا بحيث لا تظهر كل الروابط بالوزن نفسه:
  - صف/مستوى للروابط الأساسية
  - صف/مستوى للروابط الثانوية الخاصة بالتوسع
- إضافة رابط مباشر إلى `Site Index` / `دليل الموقع` داخل الهيدر نفسه.
- تحسين أنماط CSS الخاصة بالهيدر لتبقى القراءة أوضح على الشاشات الكبيرة والصغيرة.
- إضافة تمايز بصري أوضح بين أنواع الصفحات المختلفة:
  - `hero--home` للصفحة الرئيسية
  - `hero--portal` للبوابات الجامعة
  - `hero--detail` للصفحات الفرعية المتخصصة
- إضافة شارات تعريفية خفيفة داخل بعض الصفحات لشرح طبيعة المسار أو وظيفته بصريًا.
- تحسين بطاقات المجالات العلمية في الصفحة الرئيسية والبوابات لتظهر بصمة أوضح من البطاقات العادية.

### تحقق تقني
- `npm run build`: ناجح بعد إضافة الأقسام والصفحات الجديدة.
- `npm run check`: ناجح بدون أخطاء أو تحذيرات أو ملاحظات.

### الحالة الحالية
- أصبح الهيكل التنفيذي للموقع أقرب بكثير إلى خريطة المشروع المعتمدة في الوثائق.
- أصبحت البنية التنظيمية للمحتوى أوضح من السابق، ولم تعد الصفحة الرئيسية مقصورة على الأقسام الأولى فقط.
- أصبحت تجربة التصفح أوضح بفضل مسارات التنقل الداخلية وصفحة الدليل العام.
- أصبح الهيدر أكثر هدوءًا وتنظيمًا بعد فصل الروابط الأساسية عن روابط التوسع.
- أصبحت الصفحة الرئيسية والبوابات والصفحات التخصصية أقل تشابهًا من الناحية البصرية.
- لا تزال أغلب الصفحات الجديدة بصياغة تأسيسية تمهيدية، وتنتظر المحتوى العلمي الحقيقي المعتمد.

### الخطوة التالية
- إدخال محتوى حقيقي معتمد في الأقسام الجديدة ذات الأولوية، أو
- توسيع الاعتماد على البنية المنظمة الجديدة في الأقسام المختلفة بدل النصوص اليدوية كلما أمكن.
